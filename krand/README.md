# krand

再現性あり・非暗号用途の疑似乱数生成器（PRNG）です。

## できること（概要）
- 再現性のある列生成（同じseedで同じ出力）
- 小さなAPI: `next_u32`, `next_f32_0_1`（[0,1)）, `range_u32(a..b)`（偏りを抑制）
- 内部は PCG32 を本体に、SplitMix64 でシード展開。範囲は Lemire 法で偏り低減。

対象外（重要）
- 暗号・セキュリティ用途には使用しないでください（鍵・トークンの生成など）。

## サンプル

### 簡単な統計

- 実行: `cargo run -p krand --example stats -- 1000000 123`
- 出力: 標本平均・母分散の推定値（期待値は mean≈0.5, var≈1/12≈0.083333）
