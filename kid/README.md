# kid

軽量なエンティティ用アリーナ（世代付きハンドル）実装。

## できること（概要）
- `Arena<T>`: 値の挿入・取得・更新・削除（O(1) 平均）。
- `EntityId`: `index` と `generation` を持つ世代付きハンドルで、古い参照の無効化を検出。
- フリーリスト再利用: 削除済みスロットを再利用し、メモリアロケーションを抑制。
- 反復処理: `iter` / `iter_mut` で占有中要素のみを走査。

## 規約（Conventions）
- 依存は標準ライブラリ（std）のみ。
- ハンドルは `(index, generation)` の組で一意性を保証。削除で `generation` をインクリメント。
- `get`/`get_mut` は世代一致時のみ参照を返す。ミスマッチや範囲外は `None`。
