# kmath

2D/3D 数学ユーティリティ。

## 規約（Conventions）
- 掛け順: `Mat * Vec`（列ベクトル）
- 座標系: 右手系、+Y は上方向
- 角度: ラジアン
- 実数型: まずは `f32` 固定（必要になれば汎用化）
- 行列合成: 右から左に適用（例: `T * R * S * v` は S→R→T の順で適用）
- 行列の内部表現: 列ベクトル前提（メモリ表現は後段で固定、現時点では API 優先）

## フェーズ1の目標
- Vec2/Vec3 の基本演算、正規化、補間
- Rect/Aabb の基礎と交差判定
- 2D 変換用 Mat3 と Transform2D（平行移動/回転/スケール）
- 充実したテスト
