# kmath

軽量な 2D/3D 数学ユーティリティ。

## できること（概要）
- ベクトル: `Vec2`/`Vec3` の加減乗除、内積、長さ、正規化、補間（`lerp`）
- 矩形: `Rect`/`Aabb2` の生成、サイズ/中心の取得、包含・交差判定、`union`/拡張
- 行列: `Mat3` による 2D 同次変換（平行移動・回転・スケール）と合成、点の変換
- 変換型: `Transform2D`（TRS）での行列化と点の変換、合成（行列として）

## 規約（Conventions）
- 掛け順: `Mat * Vec`（列ベクトル）
- 座標系: 右手系、+Y は上方向
- 角度: ラジアン
- 実数型: まずは `f32` 固定（必要になれば汎用化）
- 行列合成: 右から左に適用（例: `T * R * S * v` は S→R→T の順で適用）
- 行列の内部表現: 列ベクトル前提（メモリ表現は後段で固定、現時点では API 優先）
